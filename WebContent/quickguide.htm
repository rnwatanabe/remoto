Quick Guide

Simulations can be performed in guest or logged in mode. Guest mode is the system default, but to save properties and/or simulation results in the database and retrieve them later, you have to be logged in the system. If you don't have a user account, create a new one in the Login page.

Using demonstrations (Demos)

In case you want to see the ReMoto simulator readily in action, we have provided some demonstrations that do not require interactions with the system to select specific items or set parameter values. Just follow the steps described below in order to view the results:

  • Select the Scenarios menu;
  • In the Scenario drop-down menu select the desired demonstration (Demo). The name and a brief description of this demonstration will be shown;
  • Click on the Load button. Next, click OK in the message that will appear in your screen "Load configuration? (current settings will be deleted)";
  • Now, select the Run menu and you will see the Duration and Time step used in this simulation. You do not need to change these values. So, click on the Start button and wait (you will see a progress bar at the center of your screen);
  • Finally, at the end of the simulation, the page Simulation Results will be loaded. Select the result that you want to see (e.g., Force, EMG, Soma membrane potential) and click on Generate in new window button if you want to visualize a static figure in JPEG format or in View as interactive graphic if you want to visualize results in a graphic with zoom-in and drag capabilities. The latter requires a fast internet connection.

Constructing your own structure and stimuli

  • Choose how many neural nuclei and neurons will take part in the simulation, using the Neural Elements page in the Structure menu. Click on the Apply button in order to save your choice. Please remember to use the Apply button every time you change any setting in any page. 
  • Activate the neural nuclei using the Descending Commands (Structure menu) and/or Electrical stimulation (Stimulation menu).
  • You can change the parameter values of each element of the system by selecting the Parameters submenu (in Structure menu).
  • Set the simulation Time step and Duration in the Run menu and click on the Start button.
  • After finishing the simulation, you can either create different graphics (with zoom-in and drag capabilities) as selected in the Simulation Results page or export figures and raw data for further documentation and analysis.

A PDF version of this Quick Guide can be downloaded from the Learning Center at the ReMoto Website. In a near future a complete User’s Guide will be available. In the meantime, a broad tutorial is provided in the site as a learning tool. In addition, all pre-loaded demonstrations could be a starting point for your simulations.
